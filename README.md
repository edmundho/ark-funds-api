This API takes the ARK ETF CSV files and serves each of them up as a JSON payload.

Example response:
```
[
  {
      "date": "2/25/2021",
      "fund": "ARKK",
      "company": "TESLA INC",
      "ticker": "TSLA",
      "cusip": "88160R101",
      "shares": "3498806.00",
      "market value($)": "2386955429.32",
      "weight(%)": "10.22"
  }
]
```
